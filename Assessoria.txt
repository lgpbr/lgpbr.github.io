<p><li><b>Gestão de Pessoas</b></p>

<p>Administrar ou gerenciar uma organização requer, atualmente, muito mais do que as funções de planejar, organizar e controlar. As ameaças, a concorrência, as rápidas mudanças no contexto socioeconômico requerem do gestor habilidades humanas em alto grau de refinamento, que podem ser apreendidas com a busca pela compreensão do comportamento individual e dos grupos em situação de trabalho – o que constitui o campo de estudo do comportamento organizacional.</p>

<p>Um gestor que compreenda o comportamento organizacional estará melhor preparado para conhecer as situações de trabalho, a lidar com os problemas organizacionais, e ajudar as pessoas e grupos a alcançarem seus objetivos de maneira mais eficaz possível.</p>

<p>Assim, conhecer os aspectos que influenciam o comportamento das pessoas no trabalho pode ajudar as organizações a se tornarem mais produtivas e lucrativas.</p>
<p>Para apoiar o departamento de Gestão de Pessoas, instruímos o desenvolvimento das seguintes ações: </p>
<p><b>I</b> - Formação e Desenvolvimento de Competências;</p>
<p><b>II</b> - Avaliação da Performance dos RH;</p>
<p><b>III</b> - Gestão de Talentos humanos;</p>
<p><b>IV</b> - Coaching;</p>
<p><b>V</b> - Gestão da Estrutura Organizacional;</p>
<p><b>VI</b> - Mudança Organizacional;</p>
<p><b>VII</b> - Processos de decisão;</p>
<p><b>VIII</b> - Reengenharia de Processos;</p>
<p><b>IX</b> - Cultura organizacional;</p>
<p><b>X</b> - Comunicação organizacional;</p>

<p>Clima organizacional

<p>Em ambientes competitivos e de mudanças constantes – como o atual das organizações – para promover o desenvolvimento integral, visando enfrentar seus desafios constantes, as organizações devem, além de aplicar seus esforços na satisfação do cliente externo, também direcioná-los para o cliente interno – o colaborador, considerando que sua satisfação contribuirá com a satisfação do cliente externo, gerando maiores resultados.</p>

<p>Reconhece-se assim, a importância das pessoas nesse processo e, ao estudar este papel, ingressamos no âmbito do Comportamento Organizacional, uma vez que o mesmo tem como objetivo compreender esse sistema social, denominado organização, do ponto de vista formal e informal, bem como entender os mecanismos que influenciam o comportamento individual e coletivo (ROBBINS, 2002).</p>

<p>Trabalhamos o Clima organizacional para busca de um desempenho organizacional excelente, e é fundamental que as pessoas estejam comprometidas com os objetivos e metas da organização e que estejam satisfeitas com ambiente de trabalho. Dentro desse contexto, surge o conceito de clima organizacional, que se refere ao ambiente que envolve todos os participantes da empresa, e que está intensamente relacionado com o grau de motivação de seus participantes – à percepção daqueles aspectos que promovem diferentes comportamentos nos indivíduos integram a organização.</p>

<p>Assim, o Clima Organizacional constitui o meio interno de uma organização, a atmosfera psicológica e característica que existe em cada uma delas.</p>


<p><b>Equipes de alta Performance</b></p>

<p>À medida que o ambiente de competitividade e globalização atinge as organizações/instituições, aumentam as demandas e as pressões, e estruturas de trabalho e negócios tradicionais precisam se transformar gradativamente em sistemas baseados em equipes.</p>

<p>Utilizar um sistema baseado em equipes não constitui uma transformação radical, uma vez que os empregados sempre tiveram contato entre si; mas desta forma é possível coordenar e aproveitar melhor suas interações, e ainda estimular as pessoas a trabalharem efetivamente para objetivos comuns.</p>

<p>A utilização de equipes de trabalho nas organizações tem se popularizado e são inúmeras as situações que podem justificar o seu uso, como trabalhos e projetos que:</p>

<li>Exigem a combinação de conhecimentos, especialidades e perspectivas que não são encontrados em um único individuo;</li>
<li>Demandam pessoas que sejam altamente interdependentes, a fim de realizar um trabalho e atingir um objetivo comum;</li>
<li>Apresentam um desafio significativo para todos os componentes; ou seja, algo que desperte o potencial de cada indivíduo e direcione suas emoções e competência no sentido desejado (REIS et al, 2005).</li>

<p>Os benefícios da utilização de equipes de trabalho são vários, dentre os quais, podemos citar:</p>

<li>Melhoria do desempenho;</li>
<li>O ambiente torna-se mais motivador;</li>
<li>A responsabilidade pelas tarefas é compartilhada;</li>
<li>Habilidade de reagir mais prontamente em face de mudanças tecnológicas;</li>
<li>Uso eficaz dos poderes delegados;</li>
<li>Compromisso compartilhado com os objetivos;</li>
<li>Aumento da criatividade e eficácia ao lidar com problemas;</li>
<li>Maior habilidade e flexibilidade ao fazer designações de tarefas;</li>
<li>Decisões mais eficazes;</li>
<li>Melhoria nas comunicações; e/ou,</li>
<li>Incremento do treinamento e desenvolvimento interfuncional.</li>

<p>Estas vantagens originam-se principalmente na sinergia das habilidades e experiência dos membros. Estamos dispostos e prontos a construir e/ou orientar sua equipe de sucesso.</p>
<p>Maximize o potencial da sua empresa, queremos ajudá-lo a crescer!</p>

    $(function() {
        $('a[href*=#]:not([href=#],[data-toggle],[data-target],[data-slide])').click(function() {
            if (location.pathname.replace(/^\//, '') == this.pathname.replace(/^\//, '') || location.hostname == this.hostname) {
                var target = $(this.hash);
                target = target.length ? target : $('[name=' + this.hash.slice(1) + ']');
                if (target.length) {
                    $('html,body').animate({
                        scrollTop: target.offset().top
                    }, 1000);
                    return false;
                }
            }
        });
    });
    //#to-top button appears after scrolling
    var fixed = false;
    $(document).scroll(function() {
        if ($(this).scrollTop() > 250) {
            if (!fixed) {
                fixed = true;
                // $('#to-top').css({position:'fixed', display:'block'});
                $('#to-top').show("slow", function() {
                    $('#to-top').css({
                        position: 'fixed',
                        display: 'block'
                    });
                });
            }
        } else {
            if (fixed) {
                fixed = false;
                $('#to-top').hide("slow", function() {
                    $('#to-top').css({
                        display: 'none'
                    });
                });
            }
        }
    });
    // Disable Google Maps scrolling

    // Disable scroll zooming and bind back the click event
    var onMapMouseleaveHandler = function(event) {
        var that = $(this);
        that.on('click', onMapClickHandler);
        that.off('mouseleave', onMapMouseleaveHandler);
        that.find('iframe').css("pointer-events", "none");
    }
    var onMapClickHandler = function(event) {
            var that = $(this);
            // Disable the click handler until the user leaves the map area
            that.off('click', onMapClickHandler);
            // Enable scrolling zoom
            that.find('iframe').css("pointer-events", "auto");
            // Handle the mouse leave event
            that.on('mouseleave', onMapMouseleaveHandler);
        }
        // Enable map zooming with mouse scroll when the user clicks the map
    $('.map').on('click', onMapClickHandler);
